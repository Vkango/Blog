# 📕 Blog

为练习前端+Node.js写的博客管理。

代码并不规范甚至有很多屎，写的也很匆忙。

欢迎优化、交流。

以 HTML + CSS + JS 为前端、Node.js & Python为后端制作的轻量级博客管理。

后端端口默认开放在5501，可自行去read-blog.js进行修改。

# ⭐ 功能

1. 添加博客、修改博客、删除博客

2. 自定义封面

3. 使用Markdown写博客，自动转换HTML
